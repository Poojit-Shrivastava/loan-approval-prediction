# Loan Approval Prediction

A Machine Learning web application that predicts loan approval status using Logistic Regression.

## Features

* Data Cleaning
* Missing Value Handling
* Exploratory Data Analysis (EDA)
* Feature Encoding
* Model Comparison
* Cross Validation
* Feature Importance Analysis
* Streamlit Web Application

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Matplotlib
* Seaborn

## Model Performance

* Test Accuracy: 78.86%
* Cross Validation Accuracy: 80.46%

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Streamlit Deployment
